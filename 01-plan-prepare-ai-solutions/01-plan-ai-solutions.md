# Plan and prepare to develop AI solutions on Azure

- Microsoft Foundry
  - [Foundry URL](https://ai.azure.com/home)
  - Unified platform for AI apps and agents
  - Foundry resources provide compute, storage, model delivery and other services
    - Platform for project
  - Foundry projects are isolated work enviornments
  - Projects contain
    - Models
      - Choose and deploy models
    - Agents
      - Create and manage agents
    - Tools
      - Connect agents to tools so they can perform actions and access information
    - Knowledge
      - Manage agent data sources so they have information to act

    - Foundry tools
      - Azure language in Foundry tools
        - Natural language processing
          - Can we detect lanaguage
          - Can we find key terms
      - Azure speech
        - Speech recognition and synthesis
      - Azure Translator
        - Multilingual text and document translation
          - Language translation
      - Azure Document Intelligence
        - Document processing and extraction
          - Read from documents and extract information
      - Azure Content Understanding
        - Multimodal content analysis 
  
- VS Code extensions for Foundry
  - Foundry Toolkit for VS Code
    - Installs Microsoft Foundry extension
